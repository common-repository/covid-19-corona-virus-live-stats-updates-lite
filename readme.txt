== Description ==
COVID-19 Corona Virus Live Stats & Updates is plugin for show update related to Corona Virus. This plugin have every thing you need related to Corona Virus updates and Stats.


**Pro Version**

[Buy Pro Version](https://codecanyon.net/item/covid19-corona-virus-live-stats-updates/26283515)

**Live preview**

[Live preview](https://ulcwp.com/all-covid-19-shortcodes/)


**Features:**

*  Global Stats [Example](https://ulcwp.com/global-updates/)
   <pre>[covid19-global-updates label="Updates From All Over The World"]</pre>
*  Country Wise Stats [Example](https://ulcwp.com/all-countries-updates/)
 <pre>[covid19-country-updates country=USA label="USA Covid19 Data"]</pre>
*  All USA State Stats [Example](https://ulcwp.com/usa-states-updates/)
    <pre>[covid19-state-updates label="USA States Covid19 Data"]</pre>
*  All countries Stats [Example](https://ulcwp.com/all-countries-updates/)
   <pre>[covid19-list-countries label="All countries Covid19 Data"]</pre>

**Pro Features:**

* Global Stats with Dark and custom color via shortcode [Example](https://ulcwp.com/global-updates/)

* Dark Layout
  <pre>[covid19-global-updates label="Updates From All Over The World" mode="dark"]</pre>

* Custom Colors 
  <pre>[covid19-global-updates label="Updates From All Over The World" colors="#04F6C3 ,#064885"]</pre>

*  Country Wise Stats with dark mode, 3 layouts and country filtration [Example](https://ulcwp.com/country-specific-updates/)
   <pre>[covid19-country-updates country=USA country_list=1 layout=1 label="USA Covid19 Data" colors="#023E73,#F91C02"]</pre>

*  World Map [Example](https://ulcwp.com/usa-states-updates/)
   <pre>[covid19-countries-map label="World Data" colors="#F03312,#1237F0" ]</pre>

*  All countries Stats [Example](https://ulcwp.com/all-countries-updates/)

<pre>[covid19-list-countries label="All countries Covid19 Data" colors="#F03312,#1237F0"]</pre>

*  All countries Stats [Example](https://ulcwp.com/all-countries-updates/)

<pre>[covid19-list-countries label="All countries Covid19 Data" colors="#F03312,#1237F0"]</pre>
